# FastAPI Inventory CRUD

A simple Inventory Management CRUD application built using FastAPI and React.

## Tech Stack

- FastAPI
- Pydantic
- SQLAlchemy
- PostgreSQL
- React

## Features

- Add Product
- View Products
- Update Product
- Delete Product
- REST API
- PostgreSQL Database

## Run Backend

```bash
uvicorn main:app --reload
```

## Run Frontend

```bash
cd frontend
npm install
npm start
```
